## 二叉树的基本概念

### 结构与概念



```java
Class Node {
	V value;
	Node left;
	Node right;
}

```

二叉树（Binary Tree） 是另外一种树型结构，它的特点是每个节点至多只有两棵子树（即二叉树中不存在度大于2的结点），并且，二叉树的子树有左右之分，其次序不能任意颠倒。

与树的递归定义类似，二叉树的递归定义如下：二叉树或者是一棵空树，或者是一棵由一个根结点和两棵互不相交的分别称为根的左子树和右子树的子树所组成的非空树。

由以上定义可以看出，二叉树中每个结点的孩子数只能是0、1或2个，并且每个孩子都有左右之分。位于左边的孩子称为左孩子，位于右边的孩子称为右孩子；以左孩子为根的子树称为左子树，以右孩子为根的子树称为右子树。

### 性质（了解）

二叉树具有下列重要特性：

**性质1**： 在二叉树的第i层上至多有2i-1个结点（i>=1）

**性质2**： 深度为k的二叉树至多有2k-1个结点，（k>=1）.

**性质3：** 对任何一棵二叉树T，如果其终端结点数位n0，度为2的结点数为n2，则 n0 = n2 + 1

**性质4**： 具有n个结点的完全二叉树的深度为 ⌊log2n⌋+1

**性质5**： 如果对一棵有n个结点的完全二叉树（其深度为 ⌊log2n⌋+1 ）的结点按层序编号（从第1层到第 ⌊log2n⌋+1 层，每层从左到右），则对任一结点i(1<=i<=n)，有：

（1）如果i=1，则结点i是二叉树的根，无双亲；如果i>1，则其双亲PARENT(i)是结点⌊i/2⌋。

（2）如果2i>n，则结点i无左孩子（结点i为叶子结点）；否则其左孩子LCHILD(i)是结点2i。

（3）如果2i+1>n，则结点i无右孩子；否则其右孩子RCHILD(i)是结点2i+1。

## 二叉树遍历

先序：任何子树的处理顺序都是，先头节点、再左子树、然后右子树

中序：任何子树的处理顺序都是，先左子树、再头节点、然后右子树

后序：任何子树的处理顺序都是，先左子树、再右子树、然后头节点

层次：按照每层节点进行打印实现

下面将会分别实现上面的算法

### 先序遍历

#### 递归遍历

递归遍历这里实现的话很简单，因为我们是按照头->左->右进行打印的，因此我们只需要进行

1. 打印节点
2. 遍历左子树
3. 遍历右子树

代码如下：

```java
public static void pre (Node head) {
        if (head == null) {
            return;
        }
        System.out.println(head.value);
        pre(head.left);
        pre(head.right);
    }
```

#### 非递归遍历

在非递归时，我们需要使用到栈结构才能进行实现，需要经过下面三个步骤：

1. 将结点放入栈中，然后弹出打印
2. 如果结点存在**右孩子**，压入右孩子
3. 如果结点存在**左孩子**，压入左孩子

重复上面三个步骤，直到所有结点全部打印完成。代码如下：

```java
public static void pre (Node head) {
        System.out.println("pre - order: ");
        if (head != null) {
            Stack<Node> stack = new Stack<Node>();
            stack.add(head);
            while (!stack.isEmpty()) {
                head = stack.pop();
                System.out.println(head.value + " ");
                if (head.right != null) {
                    stack.push(head.right);
                }
                if (head.left != null) {
                    stack.push(head.left);
                }
            }
        }
    }
```

### 中序遍历

#### 递归遍历

递归遍历这里实现的话很简单，因为我们是按照左->头->右进行打印的，因此我们只需要进行

1. 遍历左子树
2. 打印节点
3. 遍历右子树

代码如下：

```java
 public static void in (Node head) {
        if (head == null) {
            return;
        }
        in(head.left);
        System.out.println(head.value);
        in(head.right);
    }
```

#### 非递归遍历

在非递归时，我们需要使用到栈结构才能进行实现，需要经过下面三个步骤：

1. 从结点开始，遍历整个左子树，依次放入栈中
2. 当1走到最边界的时候，弹出打印当前结点
3. 当前结点右子树重复1、2

代码如下：

```java
 public static void in (Node head) {
        System.out.println("in - order:");
        if (head != null) {
            Stack<Node> stack = new Stack<>();
            while (!stack.isEmpty() || head != null) {
                if (head != null) {
                    stack.push(head);
                    head = head.left;
                } else {
                    head = stack.pop();
                    System.out.println(head.value + " ");
                    head = head.right;
                }
            }
        }
    }
```



### 后序遍历

#### 递归遍历

递归遍历这里实现的话很简单，因为我们是按照左->头->右进行打印的，因此我们只需要进行

1. 遍历左子树
2. 遍历右子树
3. 打印节点

代码如下：

```java
public static void pos(Node head) {
        if (head == null) {
            return;
        }
        pos(head.left);
        pos(head.right);
        System.out.println(head.value);
    }
```

#### 非递归遍历

##### 方式一

方式一我们是基于先序的非递归遍历进行更改的，方式如下：

1. 将结点放入栈中，然后弹出打印
2. 如果结点存在**左孩子**，压入左孩子
3. 如果结点存在**右孩子**，压入右孩子
4. 将其所有元素压入栈，然后依次弹出，即可得到对应的后序序列

代码如下：

```java
public static void pos1 (Node head) {
        System.out.println("pos1 - order");
        if (head != null) {
            Stack<Node> stack1 = new Stack<Node>();
            Stack<Node> stack2 = new Stack<Node>();
            stack1.push(head);
            while (!stack1.isEmpty()) {
                head = stack1.pop();
                stack2.push(head);
                if (head.left != null) {
                    stack1.push(head.left);
                }
                if (head.right != null) {
                    stack1.push(head.right);
                }
            }
            while (!stack2.isEmpty()) {
                System.out.println(stack2.pop().value + " ");
            }
        }
    }
```

##### 方式二

方式一中，我们可以明显的看出来对应的缺点：需要两个栈，因此方式二就是我们基于方式一的使用。步骤如下：

1. 要保证根结点在左孩子和右孩子访问之后才能访问，因此对于任一结点P，先将其入栈。
2. 如果P不存在左孩子和右孩子，则可以直接访问它；
3. P存在左孩子或者右孩子，但是其左孩子和右孩子都已被访问过了，则同样可以直接访问该结点。
4. 若非上述两种情况，则将P的右孩子和左孩子依次入栈，这样就保证了每次取栈顶元素的时候，左孩子在右孩子前面被访问，左孩子和右孩子都在根结点前面被访问。

这个思路理解起来是容易的，但是需要注意判断条件，代码如下：

```java
public static void pos2 (Node head) {
        System.out.println("pos2 - order");
        if (head != null) {
            Stack<Node> stack = new Stack<>();
            stack.push(head);
            Node current = null;
            while (! stack.isEmpty()) {
                //拿到当前的栈顶元素
                current = stack.peek();
                //当前结点左子树不为空并且没有访问过
                if (current.left != null && head != current.left && head != current.right) {
                    stack.push(current.left);
                    //当前结点的右子树不为空并且没有访问过
                } else if (current.right != null && head != current.right) {
                    stack.push(current.right);
                    //左右子树都访问过了，来到情况2，更新head
                } else {
                    System.out.println(stack.pop().value + " ");
                    head = current;
                }
            }
        }
    }
```

### 层次遍历

层次遍历其实就是宽度优先遍历，用队列可以实现：

```java
public static void level (Node head) {
        if (head == null) {
            return;
        }
        Queue<Node> queue = new LinkedList<>();
        queue.add(head);
        while (!queue.isEmpty()) {
            Node current = queue.poll();
            System.out.println(current.value);
            if (current.left != null) {
                queue.add(current.left);
            }
            if (current.right != null) {
                queue.add(current.right);
            }
        }
    }
```

#### 常见题目

求最长的宽度

##### 思路一

对于某一层的最大长度，我们可以使用一个很简单的方法：

1. 使用HashMap的kv进行存储，放入结点时放入其所在的层数。
2. 在遍历某一层时，我们将这一层的结点数进行相加，得出这一层的结点数。
3. 当遍历完某一层时，进行比较，同时将层数这些进行相加与初始。

```java
public static int maxWidthUseMap(Node head) {
        if (head == null) {
            return 0;
        }
        Queue<Node> queue = new LinkedList<>();
        queue.add(head);
        HashMap<Node, Integer> levelMap = new HashMap<>();
        levelMap.put(head, 1);
        int currentLevel = 1;
        int currentLevelNodes = 0;
        int max = 0;
        while (!queue.isEmpty()) {
            Node current = queue.poll();
            int currentNodeLevel = levelMap.get(current);
            if (current.left != null) {
                levelMap.put(current.left, currentNodeLevel + 1);
                queue.add(current.left);
            }
            if (current.right != null) {
                levelMap.put(current.right, currentNodeLevel + 1);
                queue.add(current.right);
            }
            if (currentNodeLevel == currentLevel) {
                currentLevelNodes++;
            } else {
                max = Math.max(max, currentLevelNodes);
                currentLevel++;
                currentLevelNodes = 1;
            }
        }
        max = Math.max(max, currentLevelNodes);
        return max;
    }
```

##### 思路二

我们完全可以不使用HashMap来进行存储，方法如下：

1. 在进行层次遍历加入结点时，记录最后一个结点
2. 当我们来到所记录的最后一个结点时，进行比较，同时将当前结点以及个数等重置。

```java
public static int maxWidthNoMap(Node head) {
        if (head == null) {
            return 0;
        }
        Queue<Node> queue = new LinkedList<>();
        queue.add(head);
        Node currentEnd = head;
        Node nextEnd = null;
        int max = 0;
        int currentLevelNodes = 0;
        while (!queue.isEmpty()) {
            Node cur = queue.poll();
            if (cur.left != null) {
                queue.add(cur.left);
                nextEnd = cur.left;
            }
            if (cur.right != null) {
                queue.add(cur.right);
                nextEnd = cur.right;
            }
            currentLevelNodes++;
            if (cur == currentEnd) {
                max = Math.max(max, currentLevelNodes);
                currentLevelNodes = 0;
                currentEnd = nextEnd;
            }
        }
        return max;
    }
```

### 总结

#### 对于递归的遍历

1. 先序、中序、后序都可以在递归序的基础上加工出来
2. 第一次到达一个节点就打印就是先序、第二次打印即中序、第三次即后序

## 二叉树的序列化与非序列化

Java 序列化就是指将对象转换为字节序列的过程，而反序列化则是只将字节序列转换成目标对象的过程。

这样的目的算作是可以实现对应的结构，可以看看下面这个博文：

https://www.jianshu.com/p/6219d8024d2c

### 思路：

1. 可以用先序或者中序或者后序或者按层遍历，来实现二叉树的序列化
2. 用了什么方式序列化，就用什么样的方式反序列化

基于以上两点，我们可以使用先序遍历（中序后序都一样，都是使用一样的内容），或者使用层次遍历来进行实现。

#### 方式一

使用先序遍历，同时加入null结点

```java
public static Queue<String> preSerial(Node head) {
        Queue<String> ans = new LinkedList<>();
        pres(head, ans);
        return ans;
    }

    public static void pres(Node head, Queue<String> ans) {
        if (head == null) {
            ans.add(null);
            return;
        }
        ans.add(String.valueOf(head.value));
        pres(head.left, ans);
        pres(head.right, ans);
    }

    public static Node buildByPreQueue (Queue<String> preList) {
        if (preList == null || preList.size() == 0) {
            return null;
        }
        return preb(preList);
    }

    private static Node preb(Queue<String> preList) {
        String value = preList.poll();
        if (value == null) {
            return null;
        }
        Node head = new Node(Integer.valueOf(value));
        head.left = preb(preList);
        head.right = preb(preList);
        return head;
    }
```

#### 方式二

使用层次遍历

```java
public static Queue<String> levelSerial(Node head) {
        Queue<String> ans = new LinkedList<>();
        if (head == null) {
            ans.add(null);
        } else {
            ans.add(String.valueOf(head.value));
            Queue<Node> queue = new LinkedList<>();
            queue.add(head);
            while (!queue.isEmpty()) {
                head = queue.poll();
                if (head.left != null) {
                    ans.add(String.valueOf(head.left.value));
                    queue.add(head.left);
                } else {
                    ans.add(null);
                }
                if (head.right != null) {
                    ans.add(String.valueOf(head.right.value));
                    queue.add(head.right);
                } else {
                    ans.add(null);
                }
            }
        }
        return ans;
    }

    public static Node buildByLevelQueue(Queue<String> levelList)  {
        if (levelList == null || levelList.size() == 0) {
            return null;
        }
        Node head = generateNode (levelList.poll());
        Queue<Node> queue = new LinkedList<>();
        if (head != null) {
            queue.add(head);
        }
        Node node = null;
        while (!queue.isEmpty()) {
            node = queue.poll();
            node.left = generateNode(levelList.poll());
            node.right = generateNode(levelList.poll());
            if (node.left != null) {
                queue.add(node.left);
            }
            if (node.right != null) {
                queue.add(node.right);
            }
        }
        return head;
    }

    private static Node generateNode(String value) {
        if (value == null) {
            return null;
        }
        return new Node(Integer.valueOf(value));
    }
```
